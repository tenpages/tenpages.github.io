# 中国制霸生成器
中国三十四省级行政区域制霸标记生成工具

https://lab.magiconch.com/china-ex/

## 功能
可以在线标记 居住、短居、游玩、出差、路过 标记后可生成图片进行社区分享


标记过的信息会记录到本地存储，有什么新进展再次打开网页更新即可

## 说明
简化地图在保留了地缘相接特征的基础上把省级行政区域尽可能简化成了方框，目标是把省级行政区域之间的关系通过画面展现到一目了然的程度

仅三十四省级行政区域可标记，九段线西沙群岛在地图上有标示

## 关于等级

我的个人标准

 - 居住：住过一年以上
 - 短居：住过一周以上
 - 游玩：旅行过
 - 出差：去过但完全没玩
 - 路过：汽车火车路过或飞机经停
